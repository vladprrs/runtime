**Step 1: Create `crates/sdg-loader/src/validation/schema_pass.rs`** -- Pass 1: JSON Schema conformance.

```rust
use crate::error::SdgError;
use crate::schema::create_validator;

/// Pass 1: Validate raw JSON against the embedded SDG JSON Schema.
/// Returns all schema violations found (per D-29: collect all errors within pass).
pub fn validate_schema(raw: &serde_json::Value) -> Vec<SdgError> {
    let validator = create_validator();
    validator
        .iter_errors(raw)
        .map(|error| SdgError::SchemaViolation {
            instance_path: error.instance_path.to_string(),
            schema_path: error.schema_path.to_string(),
            message: error.to_string(),
        })
        .collect()
}
```

Key detail: `error.instance_path` and `error.schema_path` are `jsonschema::paths::Location` values -- call `.to_string()` to get the JSON pointer path. Map each `ValidationError` to an owned `SdgError` immediately to avoid lifetime issues (Research Pitfall 1).

**Step 2: Create `crates/sdg-loader/src/validation/version_pass.rs`** -- Pass 2: Version compatibility.

Per D-03: SemVer version field `"schema_version": "2.0.0"`. Runtime checks major version match. Minor/patch differences accepted.

```rust
use crate::error::SdgError;
use semver::Version;

/// The major version this runtime supports.
pub const SUPPORTED_MAJOR_VERSION: u64 = 2;

/// Pass 2: Check schema_version field for SemVer compatibility.
/// Major version must match SUPPORTED_MAJOR_VERSION. Minor/patch differences accepted.
pub fn validate_version(raw: &serde_json::Value) -> Vec<SdgError> {
    let version_str = match raw.get("schema_version").and_then(|v| v.as_str()) {
        Some(v) => v,
        None => return vec![SdgError::MissingVersion],
    };

    let version = match Version::parse(version_str) {
        Ok(v) => v,
        Err(e) => return vec![SdgError::InvalidVersion {
            value: version_str.to_string(),
            reason: e.to_string(),
        }],
    };

    if version.major != SUPPORTED_MAJOR_VERSION {
        return vec![SdgError::IncompatibleVersion {
            found: version.to_string(),
            expected_major: SUPPORTED_MAJOR_VERSION,
        }];
    }

    vec![]
}
```

**Step 3: Create `crates/sdg-loader/src/validation/mod.rs`** -- Pipeline orchestrator with `load()` function.

Per D-28: Four validation passes in strict order. Per D-29: Collect all errors within each pass, halt on first failing pass.

```rust
pub mod schema_pass;
pub mod version_pass;

use std::path::Path;
use crate::error::SdgError;
use crate::types::ServiceDefinition;

/// Load and validate an SDG file from the given path.
///
/// Runs validation passes in strict order (per D-28):
/// 1. JSON Schema conformance
/// 2. Version compatibility
/// 3. Semantic validation (wired in Plan 03)
/// 4. DAG materialization (wired in Plan 03)
///
/// Each pass collects all errors. Later passes do not run if earlier pass fails (D-29).
pub fn load(path: &Path) -> Result<ServiceDefinition, Vec<SdgError>> {
    // Read file
    let content = std::fs::read_to_string(path)
        .map_err(|e| vec![SdgError::FileRead { path: path.to_owned(), source: e }])?;

    // Parse JSON
    let raw: serde_json::Value = serde_json::from_str(&content)
        .map_err(|e| vec![SdgError::JsonParse { message: e.to_string() }])?;

    // Pass 1: Schema conformance
    let schema_errors = schema_pass::validate_schema(&raw);
    if !schema_errors.is_empty() {
        return Err(schema_errors);
    }

    // Pass 2: Version compatibility
    let version_errors = version_pass::validate_version(&raw);
    if !version_errors.is_empty() {
        return Err(version_errors);
    }

    // Deserialize into typed structs
    let definition: ServiceDefinition = serde_json::from_value(raw)
        .map_err(|e| vec![SdgError::Deserialization { message: e.to_string() }])?;

    // Pass 3: Semantic validation (placeholder -- wired in Plan 03)
    // Pass 4: DAG materialization (placeholder -- wired in Plan 03)

    Ok(definition)
}

/// Validate raw JSON without loading from file.
/// Useful for testing or when JSON is already in memory.
pub fn validate(raw: &serde_json::Value) -> Result<ServiceDefinition, Vec<SdgError>> {
    let schema_errors = schema_pass::validate_schema(raw);
    if !schema_errors.is_empty() {
        return Err(schema_errors);
    }

    let version_errors = version_pass::validate_version(raw);
    if !version_errors.is_empty() {
        return Err(version_errors);
    }

    let definition: ServiceDefinition = serde_json::from_value(raw.clone())
        .map_err(|e| vec![SdgError::Deserialization { message: e.to_string() }])?;

    Ok(definition)
}
```
